# CoderBot
Discord bot that help you code! or just search the internet (possibly)

# Commands

- `:help` TO GET INFORMATION ON THE BOT

- `:the question` FOR ASKING THE CODING QUESTION

- `:news` FOR GETTING TODAYS NEWS
